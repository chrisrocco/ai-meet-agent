    - MockProvider implements RealtimeAudioProvider

    - MockProvider.disconnect() sets state to 'disconnected' and emits 'disconnected'

    - MockProvider.getState() returns current state
    - MockProvider.simulateAudio(buf) emits 'audio' event
    - MockProvider.simulateText(text) emits 'text' event

    - MockProvider.simulateLatency(ms) emits 'latency' event
